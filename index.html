    Gauß-Prozess Emulatoren als Ersatzmodelle

    • GPE: Interpolationsmethode
    • Eingabe
      • bekannte Datenpunkte \(\boldsymbol{X}^\bullet\), \(\boldsymbol{y}^\bullet\)
      • Kernel-Funktion
        \(C(\boldsymbol{x}_a-\boldsymbol{x}_b,\boldsymbol{\theta}) = \theta_0 (1+\sqrt{6}d)\exp(-\sqrt{6}d)\)
        \(\textstyle d = \sqrt{\sum_{i=1}^{n} \left(\frac{x_{a,i}-x_{b,i}}{\theta_i}\right)^2}\)
      • angepasste Koeffizienten \(\boldsymbol{\theta}\)
    • Ausgabe
      • Interpolationsfunktion \(y^\circ = \mu(\boldsymbol{x}^\circ)\)
      • Unsicherheitsfunktion \(\sigma^2(\boldsymbol{x}^\circ)\)
      • Ableitungen \(\nabla y^\circ\)

    Parallelisierter Ansatz

    1. ausgewählte Punkte → Modell
      \(\boldsymbol{\vartheta}^\bullet = \mathcal{M}(\boldsymbol{\tilde{p}})\)
    2. Zielwerte bestimmen
      \(y_{\boldsymbol{\tilde{p}}} = f_\mathrm{obj}(\boldsymbol{\vartheta}^\ast,\boldsymbol{\vartheta}^\bullet)\)
    3. Ersatzmodell(e) trainieren
      \(\vartheta^\circ = \mathcal{P}(\boldsymbol{\tilde{p}}) \approx \mathcal{M}(\boldsymbol{\tilde{p}}) = \boldsymbol{\vartheta}^\bullet\)
    4. neue Punkte vorschlagen
      \(\Delta \boldsymbol{\tilde{p}} = c \cdot \boldsymbol{e} \)
    5. Punkte auswählen
      \(\xi \cdot m^\mathrm{explore} + (1-\xi) \cdot m^\mathrm{exploit}\)
    6. wiederholen
